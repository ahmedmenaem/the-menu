# The Menu

is a react web app to display food menu

## Installing Deps

```
lerna bootstrap
```

## Running app

```
lerna run --parallel start
```

## user credentials

### admin

```

```

### normal user

```
user@the.com
123456
```

## Build app

```
docker-compose up
```

## Built With

- [NodeJS](https://reactjs.org/) - The web framework used
- [Lerna](https://github.com/lerna/lerna#getting-started) - Monorepo management tool
